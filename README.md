# ChatPSY_demo
目前虽然有类似于心理咨询的应用，但只能输出一些机械式的结果：如找专业心理咨询师，多锻炼等，这会使得用户产生不信任感，因为并没有像医院一样提供一些必要的辅助诊断。因此，我们的应用将会利用医院常用的精神量表进行打分，提供一些有参考意义的结果。
